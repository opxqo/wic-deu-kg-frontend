# 地理围栏


## 检查位置


**接口地址**:`/api/geo/check`


**请求方式**:`POST`


**请求数据类型**:`application/x-www-form-urlencoded,application/json`


**响应数据类型**:`*/*`


**接口描述**:<p>检查GPS坐标是否在校园范围内</p>



**请求示例**:


```javascript
{
  "latitude": 0,
  "longitude": 0
}
```


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|geoCheckRequest|GeoCheckRequest|body|true|GeoCheckRequest|GeoCheckRequest|
|&emsp;&emsp;latitude|||false|number(double)||
|&emsp;&emsp;longitude|||false|number(double)||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|ApiResponseGeoCheckResult|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|data||GeoCheckResult|GeoCheckResult|
|&emsp;&emsp;allowed||boolean||
|&emsp;&emsp;message||string||
|&emsp;&emsp;geoFenceEnabled||boolean||
|&emsp;&emsp;locationAllowed||boolean||
|&emsp;&emsp;ipAllowed||boolean||
|&emsp;&emsp;distanceMeters||integer(int32)||
|&emsp;&emsp;centerLat||number(double)||
|&emsp;&emsp;centerLng||number(double)||
|&emsp;&emsp;allowedRadiusMeters||integer(int32)||
|&emsp;&emsp;clientIp||string||
|error|错误码（仅错误时返回）|string||
|message|错误消息（仅错误时返回）|string||
|path|请求路径（仅错误时返回）|string||
|timestamp|时间戳（仅错误时返回）|string(date-time)|string(date-time)|
